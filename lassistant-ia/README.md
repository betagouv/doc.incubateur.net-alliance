# L'Assistant IA

{% hint style="info" %}
L'Assistant IA est pour l'instant en expérimentation fermée. N'hésitez pas à contacter l'équipe si vous souhaitez bénéficier d'un accès.
{% endhint %}

<figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

L'**Assistant IA** est la toute nouvelle application de [**LaSuite**](https://lasuite.numerique.gouv.fr/en), conçue pour accompagner les agents publics dans leurs recherches d’information et leurs échanges quotidiens.

L'Assistant IA est **open-source**, **hébergé en France** (sur une infrastructure qualifiée [SecNumCloud](https://cyber.gouv.fr/secnumcloud-pour-les-fournisseurs-de-services-cloud)), et pensé pour offrir une expérience simple, souveraine et sécurisée, en garantissant que les données ne sortent pas du territoire national.

L'Assistant IA permet de dialoguer facilement avec une intelligence artificielle basique, adaptée aux usages des agents publics :

* **Poser des questions** et obtenir des réponses immédiates
* **Consulter le web** pour enrichir les recherches
* **Analyser et rechercher des informations** dans ses documents
* **Retrouver l’historique des conversations** pour garder une continuité dans ses échanges.

Cette première version se concentre sur la simplicité et la souveraineté : il n’est pas encore possible de changer de modèle ni d’accéder à des fonctionnalités avancées.
